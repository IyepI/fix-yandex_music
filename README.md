# fixYMusic

Простое и легковесное браузерное расширение для Яндекс Музыки (Manifest V3), которое восстанавливает заблокированные функции интерфейса и вырезает навязчивую рекламу.

## Основные возможности

* **Свободный доступ к Коллекции** - исправляет боковое меню и возвращает возможность полноценно переходить в раздел «Моя коллекция» по прямой ссылке.
* **Разблокировка скролла** - возвращает прокрутку колесиком мыши (`wheel`) внутри списков треков и плейлистов, если скрипты сайта принудительно заблокировали интерфейс.
* **Умный баннерорез** - автоматически находит и полностью удаляет рекламные блоки из HTML-структуры страницы (включая верхние баннеры, боковую рекламу и динамические всплывающие окна-порталы).
* **Динамический мониторинг** - использует `MutationObserver`, благодаря чему расширение мгновенно реагирует на появление новых элементов рекламы или блокировок без перезагрузки страницы.

## Как установить в браузер (Режим разработчика)

Поскольку расширение создано для личного использования и оптимизации интерфейса, его можно установить вручную в любой браузер на движке Chromium (Яндекс.Браузер, Google Chrome, Opera, Edge):

1. **Скачайте код** проекта в отдельную папку на вашем компьютере.
2. Откройте браузер и перейдите на страницу управления расширениями (введите в адресную строку `chrome://extensions/` или перейдите через Меню -> Настройки -> Расширения).
3. В правом верхнем углу включите тумблер **«Режим разработчика»** (Developer mode).
4. В левом верхнем углу нажмите появившуюся кнопку **«Загрузить распакованное расширение»** (Load unpacked).
5. Выберите папку, в которой лежат файлы `manifest.json` и `fix.js`.
6. Перейдите на сайт Яндекс Музыки (или обновите вкладку, если она уже была открыта).
